# 安鼠之亂｜台北鼠患深度調查 — 設計構想

## 設計方向一：調查報導暗室美學（選定）

**Design Movement：** 調查新聞暗室 × 都市紀實攝影

**Core Principles：**
1. 深夜城市的緊張感——深黑底色、微弱光源，如同記者深夜追查的氛圍
2. 數據可視化作為敘事工具——數字、圖表以戲劇化方式呈現
3. 滾動敘事（Scrollytelling）——讀者跟隨故事線向下探索
4. 互動參與感——問答、投票讓讀者成為故事的一部分

**Color Philosophy：**
- 主色：深炭黑 `#0D0D0D`、深夜灰 `#1A1A1A`
- 強調色：警示橘紅 `#E84B3A`（危機感）、骯髒黃 `#C8A84B`（城市燈光）
- 文字：米白 `#F0EDE8`（主文）、灰白 `#9A9A9A`（次要文字）
- 數據高亮：毒藥綠 `#4CAF50`（生態）、血紅 `#D32F2F`（危機）

**Layout Paradigm：**
- 全版寬英雄區塊，文字左對齊，圖像右側出血
- 時間軸採垂直滾動觸發式展開
- 非對稱雙欄佈局（文字40% + 視覺60%）
- 互動問答採全寬遮罩式彈出

**Signature Elements：**
1. 紅色警示線條與數字標注（如調查報告的手寫標記）
2. 老鼠足跡圖案作為分隔元素
3. 新聞紙質感紋理背景（低透明度噪點）

**Interaction Philosophy：**
- Scroll-triggered 動畫：數據隨滾動計數
- 點擊問題按鈕展開答案（accordion 式）
- 時間軸點擊切換年份內容
- 生態鏈圖示 hover 顯示詳細說明

**Animation：**
- 英雄區塊：文字從下方淡入（0.6s ease-out）
- 時間軸節點：左右交替滑入（stagger 100ms）
- 數字計數：從0計數到目標值（1.5s）
- 問答展開：高度動畫 + 淡入（200ms）

**Typography System：**
- 標題：Noto Serif TC（粗體，具有傳統報紙感）
- 副標題：Noto Sans TC（中等粗細）
- 正文：Noto Sans TC（Regular，行高1.8）
- 數字/數據：Oswald（無襯線，強調感）

## 網站架構規劃

### 區塊一：英雄首頁（Hero）
- 全版黑色背景，台北101夜景 + 獵鷹剪影
- 大標題：「牠們在等什麼？」
- 副標題：台北101上空的獵鷹，揭開一場城市生態危機
- 滾動提示箭頭

### 區塊二：事件起點（The Spark）
- 2026年1月漢他病毒死亡事件
- 記者視角：「那是台灣25年來第一個漢他病毒死亡案例」
- 數據卡片：確診數、死亡數

### 區塊三：時間軸（Timeline）
- 互動式時間軸：2026年1月 → 5月
- 點擊各節點展開事件詳情

### 區塊四：為何鼠患難滅（Why So Hard）
- 三大成因：都更、食物、氣候
- 互動問答：「你知道老鼠多久生一胎？」

### 區塊五：101上空的獵鷹（The Falcon）
- 核心視覺：獵鷹盤旋101
- 生態鏈圖示：老鼠 → 老鼠藥 → 猛禽死亡
- 關鍵數據：61%、92%

### 區塊六：老鼠藥的代價（The Cost）
- 超級老鼠風險
- 寵物受害
- Vitamin K1缺貨

### 區塊七：互動問答（Quiz）
- 3-4題互動問題，測試讀者對鼠患的了解

### 區塊八：解決方案（Solutions）
- 國際案例：紐約、新加坡、日本
- 台灣可行方案
- 市民可以做的事

### 區塊九：尾聲（Epilogue）
- 記者結語
- 分享按鈕
